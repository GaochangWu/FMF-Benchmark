# Place the dataset here
You should place the downloaded FMFBenchmarkV2 dataset in this "data" folder. 

Before running, make sure your project structure is as follows:
```
FMF-Benchmark
├── configs
├── data
│   ├── README.md
│   └── FMFBenchmarkV2
│       └── pixel-level
│           ├── currents
│           ├── timestamps
│           ├── videos
│           ├── test.json
│           └── train.json
├── FMF
├── tools
├── LICENSE
├── README.md
├── requirements.txt
├── setup.py
└── state_dict_vicu_cls.pth
```
